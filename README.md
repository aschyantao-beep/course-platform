# 🎓 海外运营培训课程教学网站

一个完整的在线课程教学平台，专为海外运营培训设计。

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）
1. 双击 `启动网站.bat` 文件
2. 等待服务器启动完成
3. 在浏览器中访问 http://localhost:8000

### 方法二：手动启动
1. 打开命令行/终端
2. 进入项目目录
3. 运行命令：`python -m http.server 8000`
4. 在浏览器中访问 http://localhost:8000

## 📋 功能概览

- ✅ **10门完整课程** - 涵盖海外运营各个方面
- ✅ **学习进度追踪** - 自动记录学习时间和进度
- ✅ **在线测验系统** - 课后测试巩固知识
- ✅ **管理员后台** - 完整的内容管理功能
- ✅ **响应式设计** - 支持手机、平板、电脑访问

## 📱 访问方式

### 本地访问
- 电脑: http://localhost:8000
- 手机/平板: http://你的电脑IP:8000

### 局域网访问
其他设备可通过你的电脑IP地址访问，例如：
- http://192.168.1.100:8000

## ⚠️ 注意事项

1. **Python环境**: 确保电脑已安装Python 3.x
2. **防火墙**: 如需局域网访问，请允许8000端口通过防火墙
3. **文件完整性**: 请保持所有文件在同一目录下
4. **浏览器**: 建议使用Chrome或Firefox获得最佳体验

## 🔧 故障排除

### 无法启动服务器
- 检查Python是否正确安装
- 确认8000端口未被占用
- 尝试使用其他端口：`python -m http.server 8080`

### 页面显示异常
- 清除浏览器缓存
- 检查所有文件是否完整
- 确认网络连接正常

### 图片无法显示
- 检查assets文件夹中的图片文件
- 确认图片文件名与HTML中的引用一致
