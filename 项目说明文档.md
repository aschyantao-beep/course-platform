# 海外运营培训课程教学网站

## 项目概述
这是一个完整的在线课程教学网站，包含前端展示、管理后台、课程管理、测验系统等功能模块。

## 文件结构说明

### 📁 根目录文件
- `index.html` - 网站首页，展示课程列表和网站介绍
- `course-detail.html` - 课程详情页面模板
- `quiz.html` - 在线测验页面

### 📁 admin/ - 管理员后台
- `login.html` - 管理员登录页面
- `dashboard.html` - 管理员仪表板
- `js/`
  - `admin-login.js` - 管理员登录逻辑
  - `dashboard.js` - 仪表板功能脚本

### 📁 courses/ - 课程页面
包含10个独立的课程页面：
- `course1.html` 到 `course10.html` - 各个课程的详细内容页面

### 📁 assets/ - 课程图片资源
- `course1.jpg` 到 `course10.jpg` - 对应各课程的封面图片

### 📁 css/ - 样式文件
- `style.css` - 主要网站样式
- `admin.css` - 管理后台样式
- `course-detail.css` - 课程详情页样式
- `quiz.css` - 测验页面样式

### 📁 js/ - JavaScript功能脚本
- `main.js` - 主要网站功能脚本
- `course-detail.js` - 课程详情页交互功能
- `course-progress.js` - 课程进度跟踪
- `learning-timer.js` - 学习时间计时器
- `learning-tracker.js` - 学习行为追踪
- `quiz.js` - 测验系统逻辑

### 📁 images/ - 其他图片资源
（当前为空，可用于存放网站其他图片）

### 📁 public/ - 公共资源
（当前为空，可用于存放公共静态资源）

### 📁 src/ - 源代码
（当前为空，可用于存放源代码文件）

## 功能特性

### 🎯 核心功能
1. **课程展示** - 10门完整的海外运营培训课程
2. **学习追踪** - 自动记录学习进度和时间
3. **在线测验** - 课程配套测试系统
4. **管理后台** - 完整的管理员控制面板

### 🔧 技术特性
- 纯HTML/CSS/JavaScript实现
- 响应式设计，支持多设备访问
- 本地存储学习数据
- 模块化代码结构

## 使用说明

### 启动网站
1. 确保所有文件在同一目录下
2. 使用Python启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问：`http://localhost:8000`

### 管理后台访问
- 访问路径：`/admin/login.html`
- 通过管理员账号登录后可访问仪表板

## 部署建议

### 本地开发
- 使用Python HTTP服务器进行本地测试
- 修改代码后刷新浏览器即可看到效果

### 生产环境
- 可部署到任何支持静态网站的服务器
- 推荐使用Nginx或Apache作为Web服务器
- 可考虑使用CDN加速图片资源加载
